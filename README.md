# Sistema Distribuído de Transações Financeiras

🚧 **Status: Em Desenvolvimento** 🚧

Este repositório contém o desenvolvimento de um sistema distribuído projetado para simular transações de investimentos financeiros, focando na consistência de dados em ambientes descentralizados. 

A arquitetura utiliza o padrão **Saga** para gerenciar transações através de múltiplos microsserviços independentes.

## Stack Tecnológica Principal
* **Linguagem:** Java
* **Framework:** Spring Boot
* **Mensageria:** RabbitMQ
* **Infraestrutura:** Docker
